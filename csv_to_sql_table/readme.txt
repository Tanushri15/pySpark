How to download public dataset from Kaggle?

Step 1 :
Create Kaggle Account onto https://www.kaggle.com

Step 2 :
Once login to Kaggle platform ---> Go to Settings

Step 3 :
Scroll dowm to find API section under accounts.

Step 4:
Click on "Create New Token"

-- A json file will get download that contain your username and key that we will use at the time of downloading datasets from Kaggle
